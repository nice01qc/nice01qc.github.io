#### javac 编译时关于指定jar包问题

```bash
#编译1，指定文件目录
javac -Djava.ext.dirs=D:/javacode/lib Nice.java
#运行1
java -Djava.ext.dirs=D:/javacode/lib Nice

#编译2，指定jar包
javac -cp D:/javacode/lib/myjar.jar; Nice.java
#运行2
java -cp D:/javacode/lib/myjar.jar; Nice
```
